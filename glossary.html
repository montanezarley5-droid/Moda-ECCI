Technical Glossary

A curated dictionary of key terms bridging fashion, design, and frontend development. Explore 15 terms across five disciplines.

Term (EN) Traducción (ES) Technical Definition Category
Colorway Combinación de Color A specific combination of colors applied to a garment, textile, or digital product. In fashion, it determines how a pattern is rendered across a collection; in UI, it refers to a palette variant applied to a component system. Fashion
Pantone Matching System Sistema Pantone A standardized color identification system used across industries to ensure color consistency regardless of medium or geography. Each color has a unique PMS number enabling precise replication in printing, textiles, and digital screens. Design
Sustainable Fashion Moda Sostenible A design and production approach that prioritizes environmental responsibility, ethical labor, and circular economy principles. It encompasses material selection, supply chain transparency, and end-of-life garment management. Fashion
Design Token Token de Diseño A named variable that stores visual design decisions (colors, spacing, typography, shadows) in a format consumable by both design tools and code. Tokens enable consistent, scalable design systems across platforms and products. Frontend
Glassmorphism Glassmorfismo A UI design trend characterized by frosted-glass aesthetics using CSS backdrop-filter blur, semi-transparent backgrounds, subtle borders, and layered depth. It creates a sense of dimensionality and modernity in digital interfaces. Frontend
Visual Hierarchy Jerarquía Visual The intentional arrangement of visual elements to guide the viewer's eye according to importance. Achieved through variations in size, contrast, color, spacing, and position. Critical in both fashion editorial layout and UI/UX design. Design
Responsive Design Diseño Responsivo A web development approach where layouts, images, and interfaces adapt fluidly to different screen sizes and device types using CSS media queries, flexible grids, and relative units. Ensures consistent experiences from mobile to desktop. Frontend
Color Psychology Psicología del Color The study of how colors influence human perception, emotion, and behavior. Red evokes urgency, blue builds trust, green suggests nature and balance. Applied in branding, fashion, and interface design to shape user experience subconsciously. Design
User Experience (UX) Experiencia de Usuario The totality of a person's perceptions and responses resulting from using a product or service. UX encompasses usability, accessibility, emotional response, and interaction quality — the equivalent of the "feel" of wearing a well-tailored garment. UX
Upcycling Supraciclaje The process of transforming waste or discarded materials into new products of higher quality or value. In fashion, it involves repurposing old garments into new pieces. In tech, it parallels refactoring legacy code into modern, maintainable architecture. Fashion
Semiotics Semiología The study of signs and symbols and their use or interpretation. In fashion, garments function as sign systems communicating cultural codes and identity. In design, semiotics informs how icons, colors, and layouts communicate meaning to diverse audiences. Design
Intersection Observer API API Intersection Observer A browser JavaScript API that asynchronously observes changes in the intersection of a target element with an ancestor element or the viewport. Widely used to implement performant scroll-triggered animations without main thread blocking. Frontend
Microinteraction Microinteracción A small, contained product interaction designed to accomplish a single task. Examples include hover states, like animations, loading indicators, and form validation. Microinteractions add personality and communicative feedback to digital products. UX
Brand Identity Identidad de Marca The visible elements of a brand — logo, color palette, typography, imagery, and tone of voice — that collectively distinguish it in the consumer's mind. Both fashion houses and tech products invest heavily in cohesive brand identity systems. Design
Circular Economy Economía Circular An economic model that aims to eliminate waste by keeping materials in use for as long as possible through reuse, repair, remanufacturing, and recycling. Applied in fashion to create closed-loop systems where garments are designed for multiple life cycles. Technology
Accessibility (a11y) Accesibilidad The practice of designing and developing digital products usable by people of all abilities, including those with visual, auditory, motor, or cognitive disabilities. Guided by WCAG standards, it includes color contrast ratios, semantic HTML, and keyboard navigation. UX
CSS Custom Properties Variables CSS Native CSS variables defined using the -- prefix that store reusable values (colors, sizes, etc.) accessible throughout a stylesheet. They enable dynamic theming, design token implementation, and runtime value changes via JavaScript — the backbone of modern design systems. Frontend
Draping Drapeado A fashion design technique where fabric is placed and pinned directly on a dress form or human body to create garment shapes, as opposed to flat pattern making. It requires understanding fabric behavior, gravity, and body contours — analogous to intuitive prototyping in UX design. Fashion
